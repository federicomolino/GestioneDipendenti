Inserimento Presenze da File

Per poter procedere con l'inserimento tramite file è obbligatorio procedere seguendo le regole sotto citate:

  • Campo "data": Il formato corretto è "2025-06-12"
  • Campo "modalità": Possibili valori:
    • UFFICIO
    • SMARTWORKING
    • FUORISEDE
  • Campo "stato": Possibili valori:
    • PRESENTE
    • FERIE
    • PERMESSO
  • Campi "ORE": Verifica che l'ora di entrata sia prima dell'ora di uscita (valido per stato PRESENTE o PERMESSO)
  • Campi "DATE": Verifica che la data di inizio ferie sia prima della data di fine ferie (valido per stato FERIE)
  • Campo "chiudi_giornata": Può assumere valore "0" o "1"
    • Valore "0": Giornata aperta
    • Valore "1": Giornata chiusa

    • ATTENZIONE!

    Se viene inserito lo stato FERIE i valori delle ore anche se compilati saranno cancellati.
    Se viene inserito PERMESSO o PRESENTE, i campi relativi alle date saranno cancellati.
